# AgriSight | Farmers Market Intelligence

AgriSight is an AI-powered platform designed to empower farmers and policymakers with data-driven market intelligence. 

## Key Features

- **Price Trends & Forecasting:** Uses AI-driven models to predict crop prices and provide actionable "Sell" or "Store" recommendations.
- **Trade Policy Simulator:** Analyzes the economic ripple effects of changes in tariffs, quotas, and trade agreements.
- **Buyer Matching System:** Connects farmers directly with verified local and regional buyers based on real-time location and crop availability.
- **Live Market Dashboard:** Provides aggregated real-time data from major mandi hubs across India.

Built with Next.js, Genkit AI, and Tailwind CSS.
